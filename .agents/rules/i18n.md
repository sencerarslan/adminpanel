# i18n Rules

## Genel Strateji

- **Kütüphane**: `next-intl` — App Router ile native entegrasyonu, RSC desteği.
- **Desteklenen diller**: `tr` (varsayılan), `en`
- **Locale tespiti**: Cookie → `Accept-Language` header → varsayılan locale sırası.
- **URL yapısı**: Locale prefix yok. Locale cookie ile taşınır, URL temiz kalır (`/dashboard`, `/users`).

> **Neden prefix'siz?** Admin paneli tek tip kullanıcı kitlesine (iç ekip) hizmet verdiğinden URL'de dil bilgisi taşımak gereksiz karmaşıklık yaratır. Cookie bazlı tercih yeterlidir.

## Dosya Yapısı

```
src/
  i18n/
    config.ts           # Desteklenen locale'ler ve varsayılan
    request.ts          # Server-side locale çözümleme (next-intl)
  messages/
    tr.json             # Türkçe mesajlar (kaynak)
    en.json             # İngilizce mesajlar
```

## Kurulum

```bash
pnpm add next-intl
```

```ts
// src/i18n/config.ts
export const locales = ['tr', 'en'] as const;
export type Locale = (typeof locales)[number];
export const defaultLocale: Locale = 'tr';
```

```ts
// src/i18n/request.ts
import { getRequestConfig } from 'next-intl/server';
import { cookies } from 'next/headers';
import { defaultLocale, locales, type Locale } from './config';

export default getRequestConfig(async () => {
  const cookieStore = await cookies();
  const localeCookie = cookieStore.get('locale')?.value;
  const locale: Locale =
    locales.includes(localeCookie as Locale) ? (localeCookie as Locale) : defaultLocale;

  return {
    locale,
    messages: (await import(`../../messages/${locale}.json`)).default,
  };
});
```

```ts
// next.config.ts
import createNextIntlPlugin from 'next-intl/plugin';

const withNextIntl = createNextIntlPlugin('./src/i18n/request.ts');
export default withNextIntl({ /* diğer next config */ });
```

## Kullanım — Server Component (RSC)

```tsx
// app/(dashboard)/users/page.tsx
import { getTranslations } from 'next-intl/server';

export default async function UsersPage(): Promise<React.JSX.Element> {
  const t = await getTranslations('users');
  return (
    <main>
      <h1>{t('title')}</h1>
    </main>
  );
}
```

## Kullanım — Client Component

```tsx
'use client';
import { useTranslations } from 'next-intl';

export function DeleteButton({ name }: { name: string }): React.JSX.Element {
  const t = useTranslations('users');
  return <button>{t('deleteConfirm', { name })}</button>;
}
```

## Dil Değiştirme

```ts
// lib/locale.ts — server action
'use server';
import { cookies } from 'next/headers';
import { type Locale, locales } from '@/i18n/config';

export async function setLocale(locale: Locale): Promise<void> {
  if (!locales.includes(locale)) return;
  const cookieStore = await cookies();
  cookieStore.set('locale', locale, {
    httpOnly: false, // client'tan okunabilmeli (UI toggle için)
    sameSite: 'lax',
    path: '/',
    maxAge: 60 * 60 * 24 * 365, // 1 yıl
  });
}
```

```tsx
// components/layout/LocaleSwitcher.tsx
'use client';
import { useTransition } from 'react';
import { setLocale } from '@/lib/locale';
import type { Locale } from '@/i18n/config';

const flags: Record<Locale, string> = { tr: '🇹🇷', en: '🇬🇧' };
const labels: Record<Locale, string> = { tr: 'Türkçe', en: 'English' };

export function LocaleSwitcher({ current }: { current: Locale }): React.JSX.Element {
  const [isPending, startTransition] = useTransition();

  const toggle = (): void => {
    const next: Locale = current === 'tr' ? 'en' : 'tr';
    startTransition(async () => { await setLocale(next); });
  };

  return (
    <button onClick={toggle} disabled={isPending} aria-label="Dil değiştir">
      {flags[current]} {labels[current]}
    </button>
  );
}
```

## Zod Validasyon Mesajları

Form validasyon hataları da çevrilmelidir. Zod mesajlarını `useTranslations` ile bağla:

```ts
// hooks/useZodMessages.ts
'use client';
import { useTranslations } from 'next-intl';
import { z } from 'zod';

export function useUserSchema() {
  const t = useTranslations('validation');

  return z.object({
    name: z.string().min(2, t('minLength', { min: 2 })).max(100, t('maxLength', { max: 100 })),
    email: z.string().email(t('email')),
  });
}
```

## Kurallar

- **Hardcoded string yasağı**: Tüm UI metinleri `messages/*.json` içinde tanımlanır. Bileşen içine Türkçe/İngilizce string yazılmaz.
- **Anahtar standardı**: `namespace.camelCase` formatı (`users.deleteConfirm`).
- **Kaynak dil**: `tr.json` kaynaktır; her yeni anahtar önce Türkçeye eklenir, ardından `en.json`'a.
- **Parametreli mesajlar**: `{name}` syntax ile interpolasyon yapılır.
- **Tarih/sayı formatı**: `next-intl` formatters kullanılır:
  ```ts
  const format = useFormatter();
  format.dateTime(date, { dateStyle: 'medium' }); // TR: 27 Şub 2026
  format.number(1234.56, { style: 'currency', currency: 'TRY' }); // ₺1.234,56
  ```

Dil tercihini görebilmek için yeni env değişkeni gerekmez — cookie yeterlidir.
